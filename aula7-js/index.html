#Introdução

O que é:

  • Linguagem de programação
  • Client-Side (!)
  • Nativo nos navegadores
  • Não é Java
  • Também Server-Side, com o NodeJS
  • Linguagem de programação dinâmica (tipagem fraca)
  • Criado por Brendan Eich em 1995

Linguagem de programação, como qualquer outra. Porém foi criada para a Web e roda nativamente nos navegadores.

O que faz:

Document Object Model

DOM: API para Javascript

  • Adiciona, alterar e remover completamente elementos HTML
  • Adiciona, alterar e remover completamente atributos HTML
  • Reagir a elementos HTML
  • Adiciona, alterar e remover completamente eventos HTML
  • Validação de dados, tratamentos dinâmicos de elementos HTML (esconder/mostrar campos, por exemplo), cálculos, lógicas de decisão, entre muitos outros não são possíveis apenas com o uso de HTML

Nos permite: buscar valores que o usuário fez input, responder a eventos em controles HTML, alterar estilos (CSS) dinamicamente de elementos

Logo, se podemos obter os valores que usuário digita em um input, por exemplo, podemos criar lógicas de negócio (através de variáveis, funções, etc…) e criar de fato, sistemas web!

Navegadores nos dão acesso ao DOM através de uma variável global, existente em TODOS os navegadores, chamada document

Como faz:

DOM: Document Object Model para HTML

  • Elementos HTML em forma de Objetos
  • Disponibiliza atributos de todos os elementos HTML
  • Disponibiliza métodos de todos os elementos HTML
  • Disponibiliza eventos de todos os elementos HTML

Como inserir no documento HTML:

  • Internamente com a tag Script
  • Externamente com um arquivo de extensão .js

#Dados e variáveis

Em javascript, temos os seguintes tipos de dados nativos:

  • number: Dados numéricos, com ou sem casas decimais
  • string: Conjuntos de caracteres (texto)
  • boolean: Dados booleanos (true / false)
  • object

Em javascript temos alguns tipos de valor especiais como:

  • null: Nulo (ausência de valor)
  • undefined: Não definido
  • NaN: Not a number

Podemos declarar variáveis globais através da palavra chave var

Podemos declarar variáveis com escopo definido através da palavra chave let

Podemos declarar constantes (não podem ter seu valor alterado) através da palavra chave const

Exercício 1

Crie uma página que contenha:

  • 3 Inputs
  • 1 Botão
  • 1 Div vazia para guardar um resultado

Faça uma função Javascript que, ao clicar no botão, some os 3 valores dos inputs e mostre o resultado na Div vazia.

Exercício 2

Crie uma outra página que contenha:

  • 3 Inputs
  • 1 Div vazia para guardar um resultado
  • SEM Botão

Faça uma função Javascript que, sempre ao alterar um dos inputs, some os 3 valores dos inputs e mostre o resultado na Div vazia.

Exercícios de casa (pode ser uma página para todos eles, se preferirem):

Utilizando inputs e botões, realize os desafios abaixo:

  1. Crie uma função chamada imprimePares, que não recebe nenhum parâmetro, e que imprime todos números pares entre 1 e 100 no HTML ao clicar em um botão;
  2. Crie uma função chamada calculaMedia, que pegue 3 notas de inputs, e que imprima no HTML a média aritmética dos números
